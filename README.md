### React+Webpack+ES6开发模式脚手架
--------------------------------
在确保自己已经安装了nodejs 和yeoman的情况下，否则：

  npm install --save yeoman

在自己的空项目中执行：
  yo reactpack

得到的脚手架应该是这样的：

      .
    ├── css
    │   └── main.css
    ├── index.html
    ├── js
    │   ├── app.js
    │   └── hello.js
    ├── node_modules
    │   ├── babel-core
    │   ├── babel-loader
    │   ├── css-loader
    │   ├── lodash
    │   ├── react
    │   ├── react-hot-loader
    │   ├── react-kendo
    │   ├── react-router
    │   ├── webpack
    │   └── webpack-dev-server
    ├── package.json
    └── webpack.config.js
